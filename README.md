<h1 align="center">Dingtalk Vue</h1>

## 目标与期望

基于钉钉微应用开发的实验室绩效管理系统，实现实验室的绩效、学分、论文评审管理与钉钉对接。  
主要功能有：绩效、学分申请与审核，论文评审投票及学分管理，项目开发学分管理，实验室助研金计算等。
